# mikutter_fishing.rb
mikutter で釣りをしよう！

## What is this?
mikutter で釣りをたのしむプラグインです。

## Usage
### 釣り方
設定のショートカットに[釣る]という項目ができるのでてきとーなキーを当てます。

あとはポストボックスにえさを書いてショートカットをｯﾀﾝ-！ えものがかかるのを待ちましょう。

### 釣られ方
タイムラインに流れてきた釣りツイートを選択して右クリックを押します。すると[釣られる]という項目ができているのでポチりましょう。
